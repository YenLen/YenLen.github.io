Lang Live

Lang Live is a live-streaming mobile platform features instant photo and video sharing functions, and I was a member of UI team. Collaborating with fellow designers, we established design system and component libraries to ensure UI consistency. To develop intuitive and engaging experiences for the product, I worked closely with engineers to specialize UI elements for both iOS and Android platforms.
The goal is to allow streamers to fulfill their dreams by sharing their talents, and viewers can cultivate a sense of self-worth through interaction.

Beauty Filter

Lang Live provides beauty filter for users when they are streaming. The origin beauty filter only provided 5 levels for users to adjust.
In the latest version, a more sensitive filter is provided. I was responsible for re-designing the user interface of beauty filter and the new icons.
'PK' function in streaming room

'PK' function aims to facilitate interaction between streamers and audience. By supporting streamers with gifts and helping them win the games, audience will cultivate a sense of belonging while being in a team and collaborating with others.
Live Streaming Hourly Chart

Hourly chart is a necessary element in each streaming room. It mainly tells the users who had given out more gifts.
I re-designed this chart with the latest styleguide, a clean and simple style, to show the information clearly.
Bulletin boards

Bulletin boards allow streamers to set an announcement to viewers in live streaming room. In order to fit different conditions and personal preferences, different styles and themes are provided. I was responsible for GUI design of these bulletin boards.
